Welcome to the COVID-19 RESTful API ! Built upon Node.js , express , cherrio and request-promise

Routes available on V1.0

/countries GET

Gets the list of all currently infected countries and thier stats , could be filtered by the continent.
Param Description Type Default Expected input
continent The continent name to filter countries by. String all all, europe, north-america, asia, south-america, africa, oceania

@returns JSON Object

/country GET

Gets the stats for a specific country.
Param Description Type Default Expected input
id The id of the country , could be a country name , alpha2 or alpha3. String Required Required

@returns JSON Object

/summary GET

Gets the overall stats (continents and worldwide) for COVID-19.
Param Description Type Default Expected input

@returns JSON Object
